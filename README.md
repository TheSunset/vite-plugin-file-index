# vite-plugin-file-index

![npm version](https://img.shields.io/npm/v/vite-plugin-file-index)
![npm unpacked size](https://img.shields.io/npm/unpacked-size/vite-plugin-file-index)
![npm download](https://img.shields.io/npm/dt/vite-plugin-file-index)

> 使用场景：vite作为开发服务器，项目为多个html文件。插件生成文件索引目录页面，方便导航到各个html文件
> 仅作为本地的一些功能测试开发使用。

## 安装

```bash
npm install vite-plugin-file-index --save-dev
```

## 使用
```js
import vitePluginFileIndex from "vite-plugin-file-index";

plugins: [vitePluginFileIndex()]
```
